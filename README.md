OVERVIEW

The app supports:

Adding, completing (strike-through), deleting, and reordering tasks
Storing tasks in browser localStorage so they persist on refresh
A fully functional, responsive input field and task list UI
A contact form with first name, last name, email, and message input
Navigation between the To-Do List and Contact pages using React Router

Technology supported:

HTML
JavaScript 
React
React Router DOM
CSS

Enhancements: (Future)
(Functionality) – Edit existing tasks, add due dates, categorize by tags or priority
(UI/UX) – Responsive design, dark mode toggle, drag-and-drop task reordering
(Communication) – Contact form confirmation modal, input validation
(AI & Smart Features) – Suggested tasks based on time of day, auto-categorization
(Productivity Tools) – Daily reminders, completion streak tracking, sorting/filtering
